Code Help

You can use code for the hardware interface to process. It uses standard javascript.

This means ultimate control, and you can do anything you like with the scope, as long as it will run in it's code sandbox.

Your code's return value is 0-5, representing voltage, run in the fastest loop possible. Anything else you want to do is up to you, including slowing down reads with delays, triggering pins, etc.

Arduino Functions Available

Pin numbers start at 0.

  • analogRead(pin) - read an analog pin.
  • analogWrite(pin, value) - write to an analog pin. value is 0-255.
  • digitalRead(pin) - read a digital pin.
  • digitalWrite(pin, value) - write to a digital pin. value is 0 or 1.
